# Python_HandsOn
このリポジトリはPythonおよび機械学習周りの初学者の導入のために作成しています。
対象者として、概ね以下のレベルを想定しています。
- なにがしかコードを書いたことがある
- データの型についての理解がある
  - intとfloatとstrが違うものであることは知っていてほしい
- 分岐(if文)と繰り返し(for文)の概念を知っている

また、実行環境としてはGoogle Colaboratory(もしくはローカルのJupyter環境)を想定しています。

## Contents
1. Pythonの基本的な動作
- 数値・変数・文字列の扱い
- 配列的なものの操作
- 繰り返しと分岐
